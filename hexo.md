## 一、安装

`npm install -g hexo-cli`

## 二、建站

`$ hexo init <folder>`
`$ cd <folder>`
`$ npm install`

<!--more-->
## 三、生成静态页面

`hexo generate`

## 四、启动本地服务，进行文章预览调试

`hexo server`

## 五、每次部署的步骤

`hexo clean`
`hexo generate`
`hexo deploy`

## 六、写作
`hexo new [layout] <title>`
